SwindleStones

Welcome to SwindleStones

Each player has a set number of dice each which they roll. Players will then bet on who's dice will be higher. 

Problem Statement: 
        Write a program that rolls 4 dice for two seperate players. Name each player.
        Record each die outcome, and display the dice image. 
        Then, compare the results of Player 1 and Player 2, and determine the winner. 

        Nouns: program, 4 dice, players, outcome, image, results
        Verbs: rolls, name, record, display, compare, determine

        Defining Diagram:
        INPUTS:
          Player1 Name
          Player2 Name


        PROCESSING:
          Prompt users for Player1 name and Player2 Name
          insert player1 and player2 name into button and output

          Create 4 dice for each player and generate random 1-6 number for each die, then append corresponding image. 
          display dice outcome and image

          Create function to roll each players dice with button click, then store values into an array. 

          Create function to compare arrays and determine winner.
          display winner

          


        OUTPUTS:
          player1 four different dice results and images
          player2 four different dice results and images
          winner outcome

        Pseudocode:
        PlayerNameEntry
          declare variable player1 and player2
          prompt user for player1 and player2 Name
          display player1 and player2 into html and into each players button

        Player1FirstSixSidedDie

          declare p1DiceRoll1

              display outcome with player name and correct image

            ELSE IF roll = 3 THEN
              display outcome with player name and correct image
              return p1DiceRoll1
            ELSE IF roll = 4 THEN
              display outcome with player name and correct image
              return p1DiceRoll1
            ELSE IF roll = 5 THEN
              display outcome with player name and correct image
              return p1DiceRoll1
            ELSE roll = 6 THEN
              display outcome with player name and correct image
              return p1DiceRoll1
            ENDIF
        END
            

          RollPlayer1Dice
            declare variables for d1, d2, d3, and d4
            store d1, d2, d3, d4 from function results
            declare p1DiceArray
            return p1DiceArray into html
          END
            

          CompareDiceValues
            declare variables from html for player1 and player2
            declare p1DiceArray and p2DiceArray from html
            split dice arrays to create seperate values
            declare p1DiceWins and p2DiceWins
            sort arrays to compare p1 and p2 results
              FOR each value in array, check who won, then increment to next value
                IF p1 wins, add 1 to p1DiceWins
              ELSE add 1 to p2DiceWins
              ENDFOR
            compare dice wins
              IF p1 dice wins > p2 dice wins THEN
                display p1 winner
              ELSE display p2 winner
              ENDIF
          END
